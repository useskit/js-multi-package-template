> multiple package template for javascript

# 🛠️ Install
```
git clone https://github.com/useskit/js-multi-package-template.git
```

# ✍️ Usage
To build the projects in this repo, try these shell commands:
```
pnpm install -g @microsoft/rush
rush install
rush build
```
